## Hiremate

AI-powered interview agent that conducts interviews, evaluates answers, and generates detailed candidate reports using Google Gemini and SQLite.

This codebase is a Next.js 14 app that powers the `hiremate` project on GitHub [`https://github.com/Praveenvin/hiremate`](https://github.com/Praveenvin/hiremate).

### Getting Started

1. Install dependencies:

```bash
npm install
```

2. Initialize the local SQLite database:

```bash
node scripts/init-db.mjs
```

3. Run the development server:

```bash
npm run dev
```

4. Open `http://localhost:3000` in your browser.

For more detailed flow and architecture, see `FLOW_DOCUMENTATION.md`, `SCENARIO_WALKTHROUGH.md`, and `IMPLEMENTATION.md`.
